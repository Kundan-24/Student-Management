# Student Management System

A Student Management System built using Spring Boot, Thymeleaf, Hibernate (JPA), and MySQL.

## Features

* Add Student
* Update Student
* Delete Student
* View Student List
* Admin Login

## Tech Stack

* Java
* Spring Boot
* Thymeleaf
* Hibernate / JPA
* MySQL
* Maven

## Project Structure

* Controller Layer
* Service Layer
* Repository Layer
* Entity Layer

## Database

MySQL database is used for storing student records.
